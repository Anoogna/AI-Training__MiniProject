# Voice Command Cheat Sheet

## Fleet Tracking (dispatcher, driver, admin)

- "Where is vehicle T-07?"
- "List active fleet"
- "Locate driver"

## Delivery Coordination (dispatcher, admin)

- "Assign pending deliveries"
- "List pending deliveries"

## Driver Communication (dispatcher, admin)

- "Broadcast alert all drivers"

## Warehouse Operations (warehouse, admin)

- "Confirm pick for shipment SH-102 aisle B3"
- "Check inventory at aisle A1"

## Smart Gate (gate, admin)

- "Register entry gate 2 shipment SH-102"
- "Register exit gate 2 shipment SH-102"
- "Verify shipment at gate SH-105"

## Text Fallback

If microphone is unavailable, type the same commands in the text input below the Push to Talk button.

## Chat Bot Commands (Delivery page)

- `run bot` — Auto-assign pending deliveries
- `show pending` — List pending tasks
- `broadcast <message>` — Send alert to all drivers
